Paintr

An AI that generates images

Paintr is an AI that generates images based on a given prompt. Written in JavaScript using the brain.js library, it can be accessed on a Discord server. Use the command '/paint' followed by a description of the image you would like to generate and the AI will create an image based on your prompt. The project is a collaboration between GreyWolf-Dev and TotoroGaming.
