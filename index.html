Imagine something...
Generate it!

The /paint feature allows you to generate an image based on a description you provide. Simply type in the command /paint followed by a description of the image you would like to create in the Discord server, and the AI will generate an image for you. It's a fun and easy way to create custom images without any design skills needed.

01

Write your description in the /paint command on our Discord bot and it will start the generation of your image! The first generation is random and will be upgraded.

02

The image will then be upscaled to look better because a 64x64 image is not a good result. The final result is a 512x512 image.

03

And boom! You have your image now! You can use it for everything you want, this image should not exist :)

Bad image? Good upscale!

With the /upscale command you can upscale images up to 8 times!
You can see just below the result of an upscale. There is only a problem, this AI don't like upscaling humans... But it is still a good AI :)

01

Use the /upscale command and upload your bad quality file and then run the command. The bot should answer and start the upscale process.

02

A couple seconds later, your image will then be upscaled to 4x. The last part is also a 4x upscale.

03

Then the bot will tell you that the process is finished. It will send you the start and the end image so you can see the difference.
