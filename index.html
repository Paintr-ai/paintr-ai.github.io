Paintr

Image Generation

Paintr is an AI that generates images based on a prompt. Simply use the command '/paint' followed by your description of the image you want to generate. For example: '/paint a sunset over a mountain range'.

Image Upscaling

In addition to generating images, Paintr can also upscale existing images. Use the command '/upscale' to upscale an image by a factor of 8, for images that are 200x200 pixels or smaller. Use the command '/light-upscale' to upscale an image by a factor of 4, for images that are 650x650 pixels or smaller.
